## O que editar antes de publicar

No arquivo `index.html`, edite apenas os links dos projetos, caso os nomes dos repositórios mudem.

## Como publicar no GitHub Pages

1. Crie um repositório chamado `portfolio-rafael-ads`
2. Envie todos os arquivos deste projeto
3. Vá em `Settings > Pages`
4. Em `Source`, escolha `Deploy from a branch`
5. Em `Branch`, escolha `main` e `/root`
6. Clique em `Save`

O link final ficará parecido com:

```txt
https://farrei.github.io/portfolio-rafael-ads/
```
